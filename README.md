# Modified 4-tank System: Multivariable Control Designs
---
Multivariable Control Implementation using MatLab/Simulink. Several Implementation Procedures are provided:
  * Unconstrainted Model Predictive Control (MPC)
  * Constrained MPC
  * Non-linear MPC

**Usage**
